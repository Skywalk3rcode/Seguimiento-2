# 🧠 Mogi - Asistente de Salud Mental

Mogi es una landing page desarrollada como parte de un proyecto académico, enfocada en el bienestar emocional y la salud mental. Su objetivo es presentar una solución tecnológica accesible que brinde apoyo inicial a las personas mediante inteligencia artificial.

---

## 🚀 Descripción del Proyecto

Mogi es un asistente inteligente que busca ayudar a los usuarios a:

- Expresar sus emociones  
- Gestionar el estrés y la ansiedad  
- Acceder a apoyo emocional básico  
- Mejorar su bienestar mental  

Este proyecto no reemplaza la atención profesional, pero funciona como una herramienta de acompañamiento inicial.

---
